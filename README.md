# Homebrew Tap for ASK

This is the [Homebrew](https://brew.sh/) tap for [ASK (Agent Skills Kit)](https://github.com/yeasy/ask).

## Installation

```bash
brew tap yeasy/tap
brew install ask
```

## Update

```bash
brew update
brew upgrade ask
```

## Uninstall

```bash
brew uninstall ask
brew untap yeasy/tap
```

## Available Formulae

| Formula | Description |
|---------|-------------|
| ask | Agent Skills Kit - Package Manager for AI Agent Skills |
